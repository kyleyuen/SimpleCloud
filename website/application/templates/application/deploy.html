通用服务器性能(General Server Performance)

Unixbench运行一组单独的基准测试，汇总分数，并创建一个最后的比分，以整体评估Linux系统的性能。UnixBench是原始字节的UNIX基准测试套件，更新和很多人修改了多年。 UnixBench的目的是提供一种类似Unix的系统的性能的基本指标，因此，多个测试是用于测试系统的性能的各个方面。这些测试结果进行比较，以从一个基线系统产生一个索引值，它通常更容易比​​原始分数来处理分数。整个组索引值，然后结合在一起，使该系统中的综合索引。一些很简单的图形测试包括量度系统的2D和3D图形性能。
